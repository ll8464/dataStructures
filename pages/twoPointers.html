Two Pointers Algorithm

Two pointers is an useful algorithm used in search related problems, namely, sorted array and string problems. There are several versions of the two pointers algorithm. One version is fast and slow pointers. The fast pointer is interated faster than the slow. For example, when using nested for loops, the inner loop interates faster than the outer loop. Another example is in linked lists. You can create a fast pointer by interating the head's next pointer: head.next for the slow and head.next.next for the fast pointer. In both examples, it allows you to perform comparisons such as in remove duplication problems.

From LeetCode: private void swap(char[] str, int i, int j) { char temp = str[i]; str[i] = str[j]; str[j] = temp; } public void reverse(char[] str) { int i = 0, j = str.length - 1; while (i < j) { swap(str, i, j); i++; j--; } }
